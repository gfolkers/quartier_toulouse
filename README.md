# Quartier ou commune ?

Mini-application statique compatible GitHub Pages.

## Contenu
- recherche tolérante aux fautes
- distinction quartier de Toulouse / commune Toulouse Métropole / commune hors métropole
- affichage cartographique Leaflet

## Déploiement
1. pousser le contenu du dossier dans un dépôt GitHub
2. activer GitHub Pages sur la branche `main`
3. vérifier que le fichier `.nojekyll` est présent

## Données
Les fichiers fournis dans `data/` sont des exemples minimaux.
Remplacer par tes référentiels réels pour la production.
